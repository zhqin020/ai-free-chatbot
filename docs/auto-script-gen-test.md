# 自动脚本生成功能测试与验收建议

## 1. 功能测试
- 输入 prompt，编辑 JSON 模板，生成代码，检查代码区内容是否符合预期。
- 粘贴完整 JSON 字符串，能否正确解析并生成代码。
- 嵌套结构、数组等复杂 JSON 能否正确渲染和生成。
- 一键复制/下载功能可用。

## 2. 集成测试
- 将生成的 make_chat_request_payload 代码粘贴到 examples/example_test_extract_api.py，能否直接运行并通过端到端流程。
- 生成的 prompt、ret_json 是否与界面输入一致。

## 3. 兼容性与健壮性
- 输入非法 JSON 时有无友好提示。
- 字段类型、说明、示例等信息能否正确保存和导出。

## 4. 用户体验
- 交互流程顺畅，操作直观。
- 代码区高亮、格式化美观。
